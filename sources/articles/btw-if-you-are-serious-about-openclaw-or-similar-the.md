btw if you are serious about OpenClaw or similar, the best advice I have is:

use OpenRouter (easy observability), and export your API calls to LangFuse (free plan)

do a few tasks, and read the prompts in LangFuse. Notice where it gets confused. Read reasoning traces, tool calls, etc. Review the system prompts for irrelevant jank.

This is the fundamental analysis that allows you to tweak everything else, add new skills, etc. Don’t tweak your agent in the dark!
