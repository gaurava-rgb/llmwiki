This shortcut has helped me ship way way way faster, I've been using it manually before but now I just run every Claude Code bypassing permissions

I was getting sick of answering endless questions and giving permissions, just do the job already

I'm 100x faster now

echo 'alias cc="IS_SANDBOX=1 claude --dangerously-skip-permissions"' >> ~/.bashrc

source ~/.bashrc
