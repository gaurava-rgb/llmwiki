Great list. Here's how to actually start each one today:

Claude Code

□ Install: npm install -g @ anthropic-ai/claude-code

□ Run claude in any project folder

□ First task: "Explain this codebase"

Perplexity Computer

□ Download Perplexity desktop app

□ Enable Computer mode in settings

□ First task: "Research [competitor] and summarize their pricing"

Claude Cowork

□ Open claude. ai → Install Productivity plugin

□ Add one context file about your business

□ First task: "/productivity:start"

OpenClaw (Clawdbot)

□ Install: npm install -g clawdbot

□ Run clawdbot init to set up your workspace

□ Connect to Discord/Telegram/WhatsApp

□ First task: Create your first skill in the skills/ folder

□ Add your AGENTS. md to define how your agent behaves

AI Second Brain

□ Create a Notion database OR folder

□ Save every useful AI output there

□ Tag by use case (research, writing, code)

Automation

□ Pick ONE: Zapier (easiest) or Make (more power)

□ Build one zap: "New email → Slack notification"

□ Graduate to MCP when ready

Prompt Engineering

□ Learn one framework: CO-STAR or RISEN

□ Save your best prompts in a swipe file

□ Iterate: good prompt → test → improve

Once you master the above you'll never have to worry about money again.
