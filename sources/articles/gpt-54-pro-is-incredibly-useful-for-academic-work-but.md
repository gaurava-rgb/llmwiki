GPT 5.4 Pro is incredibly useful for academic work, but coding agents like Claude Code/Codex can't typically access it

[@steipete](<https://twitter.com/steipete>)'s "oracle" solves this

ask Claude Code or Codex to read this [github.com/steipete/oracle](<https://github.com/steipete/oracle>) , install oracle, and install this skill [github.com/steipete/oracl…](<https://github.com/steipete/oracle/blob/main/skills/oracle/SKILL.md>)

now, when you invoke "oracle", it will

>
>
>
> bundle relevant parts of your code + paper into one very long file to send as an in browser attachment
>
> create a prompt optimized for GPT 5.4 Pro for whatever problem you're tackling
>
> capture the result from GPT 5.4 Pro in the browser and send it back to the coding agent in the terminal




on deep/architectural questions, GPT 5.4 Pro is an order of magnitude better than the next best option

Terence Tao reported advising an undergrad using GPT 5.2 Pro to solve an unsolved Erdos problem

I recommend invoking "oracle" at the planning stage of a project/new idea/feature

ofc, assumes you've got a ChatGPT Pro subscription, which I highly highly recommend if you've got the budget
