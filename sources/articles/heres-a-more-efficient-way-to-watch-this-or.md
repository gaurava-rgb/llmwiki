Here's a more efficient way to "watch" this - or any - online seminar

  1. Install yt-dlp ([github.com/yt-dlp/yt-dlp](<https://github.com/yt-dlp/yt-dlp>)). Ask CC/Codex for help.
  2. Ask CC/Codex to download the subtitles with yt-dlp.
  3. Ask CC/Codex to download the slides off my site.
  4. Now, ask whatever you want:

>
>
>
> summarize this talk for me
>
> given my particular research problem, what is relevant in this talk for me
>
> how would you push back against the speaker's arguments






Much more efficient!
