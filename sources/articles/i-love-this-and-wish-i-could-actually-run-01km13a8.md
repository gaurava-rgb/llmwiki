I love this and wish I could actually run / execute the skills in browser. I think my ideal kind of “GitHub for skills” (which is more like a replit for skills) would look something like:

  * be able to see the source files
  * beautiful presentation + docs + search
  * actual execution environment attached (e.g. hosted Claude code instance)
  * editor / environment specifically focused on working on the skill



Maybe it’s a transient product in the next 2 years but as useful as people sharing raw prompts are it’s 10x more useful to have a test environment built in
