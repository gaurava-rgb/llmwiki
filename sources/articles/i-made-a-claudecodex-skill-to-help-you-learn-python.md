I made a Claude/Codex skill to help you learn Python.

Here's how it works:

>
>
>
> You invoke with /python-learning-coach.
>
> The skill stores in memory your current level of Python/programming understanding and continuously updates it. So, it always knows the level at which to give you explanations.
>
> The skill also stores in memory summaries of every day's interactions with the skill, and loads in your last few days of conversations into its context every time you use it. This helps it give more relevant advice.




This is a now common architecture shared by many memory systems, including those of OpenClaw, and which also can be used for arbitrary "learning/teaching" skills.

If you want to install it, point your coding agent at <https://t.co/9x4Xw70EK9> and ask it to help you install it.

Or, look at the README for various other installation options: <https://t.co/sbYtaO6hBa>
