Obsidian fixes it.

Just a folder with structure that the agent reads as context.

The symbiosis is simple. Humans are too lazy to manually create and link hundreds of Markdown notes - that's not an insult, that's just true.

>
>

You get a thinking partner that remembers everything. The agent gets a brain worth using.

**Step 1: One shared folder.**

Download Obsidian, create a Vault - its just a local folder.

Put all your AI projects inside it. Open terminal, navigate to that folder, launch Claude Code from there.

Same directory. Zero sync issues.

**Step 2: The prefrontal cortex -[claude.md](<http://claude.md/>)**

Create one file in the root: [claude.md](<http://claude.md/>).

Write one core rule: all Markdown files must follow Obsidian conventions and syntax.

This file is the agent's frontal lobe. It checks it constantly to understand how you think and what standard to save files in.

Extend it with /init or grab ready-made Obsidian skill repositories online.

**Step 3: Stop formatting manually.**

Dump raw input - voice notes, NotebookLM extracts, messy thoughts.

The agent converts it to clean Markdown, adds tags, links it to your past projects.

You think out loud. It structures.

**Step 4: The self-improvement loop.**

After a month, run this prompt: "Analyze all my Obsidian notes, compare them with [claude.md](<http://claude.md/>), and update the instructions to match how I actually work"

**Your assistant evolves with you. Not frozen at day one.**

## **The foundation: one folder to rule them all**

Create a folder. Call it Workspace, Brain, whatever. Open your AI agent terminal inside it. Then open Obsidian - don't create a new vault, just point it at that same folder.

That's it. Both systems now look at the same files. When the agent creates a note - Obsidian shows it instantly. When you write something in Obsidian - the agent can read it next time you ask.

## **The terminal inside your notes**

Switching between Obsidian and a terminal window sounds like nothing. It's actually death by a thousand cuts.

Install the Terminal plugin in Obsidian Community Plugins. Open it in integrated mode. Now your agent lives as a tab inside your notes - same window, same context, zero friction.
